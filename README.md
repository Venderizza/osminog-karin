# «Осьминог» — статический сайт ресторана

Цель проекта — создать современный минималистичный сайт ресторана «Осьминог», визуально максимально приближенный к референсу **https://ava.rest/**: много воздуха, спокойная бежево‑белая палитра и яркий золотисто‑жёлтый акцентный блок.

## Реализовано
- Шапка (sticky) с логотипом **«ОСЬМИНОГ»** и навигацией: **О РЕСТОРАНЕ | КОМАНДА | МЕНЮ | ГАЛЕРЕЯ МЕНЮ | ГАЛЕРЕЯ | КОНТАКТЫ**
- Hero-секция с крупным фоновым изображением и «мягкими» градиентами
- Кнопки в hero: **«Забронировать стол»** (модальное окно) и **«Смотреть меню»**
- Секция «Осьминог, как символ» с концепцией и метаданными (адрес/режим)
- Секция команды (3 шеф-повара с фото и описанием)
- Акцентная жёлтая секция «МЕНЮ» с декоративными изображениями по бокам
- Галерея меню (8 фото) + lightbox (просмотр в полноэкранном режиме)
- Галерея интерьера (4 фото) + lightbox
- Секция «Доставка»
- Футер с контактами
- Плавающая кнопка WhatsApp справа снизу
- Адаптивность (мобильное меню, перестройка сеток)

## Точки входа (URI)
- **/** → `index.html`

Якоря на странице:
- `/#about`
- `/#team`
- `/#menu`
- `/#dish-gallery`
- `/#gallery`
- `/#contacts`

## Не реализовано
- Реальная отправка заявок на бронь (сайт статический; форма работает как демо)
- Админ-панель/CRM/интеграции с внешними сервисами

## Рекомендованные следующие шаги
- Подставить реальные фотографии (интерьер/блюда/команда) и актуальные контакты
- Подключить внешний CORS‑разрешённый REST API для реальной отправки заявок (если доступен без ключей)
- Добавить отдельную страницу/файл меню (PDF или отдельный раздел с позициями)
- Добавить карту (например, embed Яндекс/Google — при необходимости)

## Технологии
- HTML5, CSS3, Vanilla JS
- Шрифты: Google Fonts (Inter + Cormorant Garamond)
- Иконки: Font Awesome (CDN)

## Публичные URL
- Референс дизайна: https://ava.rest/
- RESTful Table API (доступен в среде проекта): `tables/{table}` (в текущей версии не используется)

## Модели данных / хранилище
- В текущей версии таблицы не используются; данных, требующих хранения, нет.
